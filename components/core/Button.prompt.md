Pill-shaped athletic action button — use for any primary/secondary CTA ("WATCH THE TAPE", "GET IN TOUCH").

```jsx
<Button variant="primary" size="lg">Watch the tape</Button>
<Button variant="accent" hard>Get in touch</Button>
<Button variant="outline" size="sm">Schedule</Button>
```

Variants: `primary` (royal), `secondary` (ink), `accent` (cyan), `outline`, `ghost`. Sizes `sm|md|lg`. `hard` adds the varsity offset shadow; `full` stretches width; `as="a"` renders a link.
