Surface container. Use `hard` for the signature varsity sticker look, `ink`/`royal` for dark or brand-fill panels.

```jsx
<Card variant="plain">…</Card>
<Card variant="hard">Featured stat block</Card>
<Card variant="ink" pad={32}>Dark section content</Card>
```

Variants: `plain` (soft shadow), `hard` (offset shadow + ink border), `ink`, `royal`. `pad` sets padding px.
