The hero stat block — a huge condensed numeral with an uppercase label. Core to every recruiting artifact.

```jsx
<StatCard value="1.82" label="ERA" />
<StatCard value="65" unit="MPH" label="Fastball" variant="royal" align="center" />
<StatCard value="12" label="K per 7" variant="hard" />
```

`variant`: `plain | hard | ink | royal`. Always pair `value` with a `label`; use `unit` for MPH/K.
