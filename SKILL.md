Read the `readme.md` file within this skill, and explore the other available files.

If creating visual artifacts (recruiting pages, social graphics, flyers, mocks, throwaway prototypes, etc), copy assets out and create static HTML files for the user to view. If working on production code, you can copy assets and read the rules here to become an expert in designing with this brand.

Key files:
- `styles.css` — link this one file to get all tokens + fonts.
- `tokens/` — colors, typography, spacing, effects.
- `components/` — React primitives (Button, Badge, Tag, Card, StatCard, StatLine, NumberMark).
- `ui_kits/recruiting_site/` and `ui_kits/social_kit/` — full compositions to copy from.
- `assets/photos/` — action photography for hero/background art.

Brand shorthand: athletic red + night-black pinstripes + chalk white, gold accent; Anton (display, all-caps) + Barlow (body); big loud numbers; varsity hard-shadow cards; no logo (use the type wordmark + jersey-number device).

If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.
