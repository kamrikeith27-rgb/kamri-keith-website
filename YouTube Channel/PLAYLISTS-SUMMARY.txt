HOW TO CREATE PLAYLISTS
================================================================================

QUICK STEPS:
1. Go to YouTube Studio
2. Click "Create" → "Create Playlist"
3. Enter playlist name (from above)
4. Enter description (from above)
5. Set to Public
6. Click "Create"
7. Click "Add Videos"
8. Search for each video title and add it

Note: Videos can be in multiple playlists. The same video will appear in both
"Hitting" and "Utility Highlights" playlists.
